Help

Whiteboard is organized in rows that you can move up and down, or delete.

  • Drag and drop widgets or containers in rows
  • When you delete a row, the following rows move up
  • Within a row, when you delete a widget or a container, the following elements move to the left

  • Use widgets from the palette to display information to and gather information from the end user
  • Configure widgets using the properties panel on the right
  • You can also create custom widgets in the UI Designer

You can group widgets into containers for layout purposes. (You can also put containers in containers).

Each container also have specific behaviors

  • Repeat content for the plain container
  • Multiple content for tabs container
  • Access to $form capabilities for form container

In the scope of a page, form, or layout, variables are used for handling business data, and for modeling the behavior of widgets.
Create variables in the Variables panel. Bind them to widget properties by clicking on the Bind button and typing the variable name in the property field.
