Help

Use the UI Designer to build pages for your business applications, forms to start process instances or execute human tasks, and application layouts that apply to all pages of an application.

  • Create pages directly in the UI Designer.
  • Start form design in Bonita BPM Studio from the appropriate contract.
  • Create layouts by duplicating the default layout available in the Bonita BPM Portal resources.
  • You can import an existing page, form, or layout to edit, or duplicate existing ones

A set of standard widgets is provided, to display information to and gather information from the end users. You can also create your own custom widgets.

The UI Designer generates standard html application code, based on AngularJS. Export pages if you want to further customize them in your favorite web development environment.

To use a page in a business application, export it from the UI Designer, import it as a portal resource, and from the relevant application, add it to the list of pages and then insert it in a navigation menu.

To use a form in a process, map it in the studio to the relevant process or human task. It will be automatically packaged in the .bar archive ready to deploy in the portal.

To use a layout in a business application, export it from the UI Designer, import it as a portal resource and map it as the layout of the relevant application.
