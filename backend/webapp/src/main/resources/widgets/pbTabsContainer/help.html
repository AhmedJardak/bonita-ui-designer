Use this container to create tabs which can be displayed over the page, form, or layout content.

The basic usage of this container is to define Default title for each tab. In this case, all tab titles will be statics and you can drag and drop any widgets you want to define the tab content.

if you want more options for each tab, refer to Advanced usage.

The usage of advance options allows you to customize default display and behaviour of tabs based on data and/or JS expressions.

This is the list of accepted options you can configure for a given tab:

  • dynamicTitle (Defaults: display "Default title" as text): In case this options is filled, it will be the tab title displayed. It can be calculated dynamically to display and render html (if allowHtml is set to true).
  • allowHtml (Defaults: false): Define if dynamicTitle will be rendered as HTML.
  • hidden (Defaults: false): Whether tab is hidden or not.
  • disabled (Defaults: false): Whether tab is clickable and can be activated.
  • cssClasses (Defaults: ""): Add css classes on tab. This css class will be attached to tab-heading html element.

"Advance tab options" property requires a JSON Object as input, to make it dynamic you can define it in an Javascript Expression or a JSON variable.

Here you can find an example of a JS expression:

return {
    tab1: {
        dynamicTitle: "<b>Dynamic tab title</b>",
        allowHtml: true,
        hidden: false,
        disabled: false,
        cssClasses: ""
    },
    tab2: {
        dynamicTitle: "Tab: " + $data.myVar
    }
};

Here we just configured 2 tabs identified as tab1 and tab2, these keys will have to be defined in each tab and will allow you to identify them. We recommend you to set a user-friendly name such as "paymentDetails" instead of keeping tab1. You can notice that it is not necessary to define all options (allowHtml, hidden, disabled and cssClasses are not defined for tab2 in this example), in this case default value will be applied.

Then, you need to set this variable as input in the property "Advanced tab options" (on the tab container panel).
