Use this widget to select a date.
To select a date and a time (e.g., to schedule a meeting), use the Date time picker widget instead.

The output of the widget (its Value) is an ISO 8601 formatted string variable with the time set to midnight UTC (e.g., 2016-12-31T00:00:00.000Z).
In addition to ISO 8601 string variables (with or without time information), you can also pass Date objects or numbers (Milliseconds since epoch) as input. However, it will change the type of the variable bound to the widget Value to a String (ISO 8601) when the user selects a date.

This widget can be used to display/update business variable attributes defined in the Business Data Model (BDM) as DATE ONLY, and based on java.time.LocalDate or as DATE (NOT RECOMMENDED), and based on java.util.Date.

The Technical date format property define the format of the values displayed in the date input. It should be a valid AngularJS date format.

To display (and not update) a date retrieved from the BDM API (or another API), you can either use this Date picker widget with Read-only property set to "yes" or use a Text, Title or a Table widget and format the value using the provided uiDate filter.

Subscription edition The content of Label, Technical date format, Placeholder properties, as well as the label of button Today are localized by default. Localized format can be edited in the asset localization.json.
The values of all properties are also used as translation keys. As a result, if you choose not to keep the default values, update the asset localization.json accordingly.
Make sure the technical format value and its translations is a valid date format.
For more information, refer to the localization help section.
